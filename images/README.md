Taruh foto profil kamu di sini dengan nama file:

- `profile.jpg` → foto profil utama (dipakai di halaman Beranda & Tentang)

## Tips
- Gunakan foto berformat **JPG** atau **PNG**
- Ukuran ideal: minimal **400x400 px**, rasio **1:1** (persegi) atau **3:4** (potret)
- Nama file harus persis: `profile.jpg`
